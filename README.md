# My personal website

In `root` folder

```bash
  cp db/config/index.js.dist db/config/index.js
```

## Start database server

### Start mongodb bin

```bash
  mongod
```

### Start express server

In `root` folder (open new terminal)

```bash
  npm run serve
```

## Start VueJS server

### Install dependencies

In `app` folder (open new terminal)

```
  npm i
```

### Start hotreload server

```
  npm run serve
```
